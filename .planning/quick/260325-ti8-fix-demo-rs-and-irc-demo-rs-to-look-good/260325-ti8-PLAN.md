Key changes from old version:
- Use textual_rs::Header instead of custom Header struct
- Use textual_rs::ListView instead of custom ChannelList/UserList

- Use textual_rs::Input instead of custom InputBar reading ctx.input_buffer
- Use textual_rs::Footer for key binding hints

TCSS stylesheet using lazeport palette:
```
IrcScreen { layout-direction: vertical; background: rgb(10,10,15); color: rgb(224,224,224); }
```
