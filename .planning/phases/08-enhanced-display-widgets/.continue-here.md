<decisions_made>

- Theme cycle key default changed from Ctrl+T to F5: Ctrl+T is intercepted at OS level on macOS (stty SIGINFO character) and Windows Terminal (new tab). F5 has no common terminal binding. Old behavior restorable via App::with_theme_cycle_key(KeyCode::Char('t'), CONTROL).
- on_theme_changed callback stored on App as Option<Box<dyn Fn(&str)>>: Simpler than event dispatch. Fires with theme name string on every cycle.
